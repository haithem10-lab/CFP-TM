CFP TM Fullstack App
instructions de démarrage:

* Backend:

Se placer dans le dossier backend :

- cd backend

Installer les dépendances :

- npm install

Démarrer le serveur :

- npm run dev

Cela démarre le backend sur http://localhost:3001

 * Frontend:

Se placer dans le dossier frontend :

- cd frontend

Installer les dépendances :

- npm install

Démarrer l’application :

npm start

L’application sera disponible sur http://localhost:3000

* Stack technique:

Frontend

React + TypeScript

Tailwind CSS

Axios pour les appels http

Zod pour la validation de formulaire

Backend

Express

TypeScript

UUID pour les identifiants de tâches

* Fonctionnalités:

- Affichage de la liste des tâches par statut (To Do et Done)

- Création d’une tâche (titre et description)

- Suppression d’une tâche

- Changement de statut via un bouton "Suivant"
